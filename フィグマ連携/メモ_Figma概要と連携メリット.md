## Figmaとは？
- Webサイトやモバイルアプリの**UI/UXデザインツール**
- ブラウザベースで動作（インストール不要）
- リアルタイム共同編集が可能
- 無料プランあり
- デザイナー、開発者、プロダクトマネージャーが利用

### 主な機能
- 画面デザイン作成
- プロトタイピング（画面遷移の再現）
- コンポーネントシステム（再利用可能なデザイン部品）
- Dev Mode（開発者向け仕様確認モード）
- Auto Layout（レスポンシブ対応）

## Claude CodeとFigmaの連携

### 連携方法
- **MCPサーバー**（Model Context Protocol）を使用
- FigmaのデザインデータをClaude Codeに橋渡しする仕組み

### セットアップコマンド
```bash
claude mcp add --transport http figma https://mcp.figma.com/mcp
```

### 連携のメリット
1. **デザインからコード自動生成** - FigmaのURLを渡すだけでコードが生成される
2. **構造データを直接読み取る** - スクリーンショットではなく、レイヤー構造・コンポーネント情報を取得
3. **双方向連携** - Figma→コード、コード→Figma（Code to Canvas機能）
4. **開発スピード大幅向上** - 数日の実装が半日程度に短縮

### ワークフロー
```
デザイナー: Figmaでデザイン作成
  ↓ (Figma MCP)
開発者: Claude Codeでコード生成・実装
  ↓ (Code to Canvas)
デザイナー: 実装結果をFigmaで確認・修正
```
